# Pharma360-End-to-End-Pharmaceutical-Analytics-Dashboard-Power-BI-
Pharma360 is a multi‑page Power BI dashboard suite built to analyze pharmaceutical operations across revenue, supply chain, logistics, finance, and inventory. It provides executives and analysts with actionable insights into performance metrics, compliance, and delivery efficiency.
